def stationarity_test(X, cutoff=0.05):
    # H_0 in adfuller is unit root exists (non-stationary)
    # We must observe significant p-value to convince ourselves that the series is stationary
    pvalue = adfuller(X)[1]
    if pvalue < cutoff:
        print('p-value = ' + str(pvalue) + ' The series ' + X.name +' is likely stationary.')
    else:
        print('p-value = ' + str(pvalue) + ' The series ' + X.name +' is likely non-stationary.')

Conclusion :

Pourquoi la stationnarit est-elle importante ?

Une srie temporelle stationnaire possde des proprits statistiques constantes dans le temps (cest--dire que la moyenne, la variance et la structure dautocorrlation ne changent pas).

Une srie temporelle non stationnaire, en revanche, a au moins une de ces proprits qui volue avec le temps (souvent la moyenne). Dans le cadre de la modlisation financire, de nombreuses sries de prix montrent des tendances ou une volatilit variable, ce qui les rend non stationnaires.

Pourquoi cest important en finance quantitative ? Identifier la stationnarit est essentiel car de nombreux modles statistiques et de prvision (comme les modles ARIMA, les tests de cointgration, etc.) supposent ou ncessitent que la srie tudie soit stationnaire.

  • Avant dappliquer un modle, on teste la stationnarit (ex : test ADF - Augmented Dickey-Fuller)
  • Si la srie nest pas stationnaire, on peut la transformer (ex : diffrenciation, log-transformation, filtrage) pour quelle le devienne.

La stationnarit est cruciale pour la cointgration Problme :

La cointgration est une proprit qui permet didentifier des relations de long terme entre plusieurs sries financires (ex : deux actifs fortement lis). Si deux sries sont cointegres, cela signifie quil existe une relation stable entre elles et que leur cart (spread) est stationnaire.

Exemple :

Supposons quon observe les prix de Binance et Deribit pour un mme contrat futur. Individuellement, ces prix peuvent tre non stationnaires (car ils suivent des tendances du march). Mais si leur spread est stationnaire, cela signifie quils voluent de manire cohrente long terme. Ce type danalyse permet didentifier des opportunits darbitrage et de pairs trading.

Solution :

On teste la cointgration en vrifiant si une combinaison linaire des deux sries est stationnaire (ex : test dEngle-Granger). Si le spread est stationnaire, cela signifie quil y a une relation structurelle forte entre les actifs signal exploitable en trading.

def fetch_symbol_closes_df(symbol_str):
    import ccxt, pandas as pd
    from functools import reduce

    symbol = symbol_str
    timeframe = '15m'
    start_date = '2025-01-01T00:00:00Z'
    exchanges = ['binance', 'deribit']
    frames = []

    for ex_id in exchanges:
        ex = getattr(ccxt, ex_id)()
        ex.load_markets()

        since = ex.parse8601(start_date)
        limit = 500
        all_ohlcv = []
        
        while True:
            ohlcv = ex.fetch_ohlcv(symbol, timeframe, since, limit)
            if not ohlcv:
                break
            all_ohlcv += ohlcv

            since = ohlcv[-1][0] + 1  
            if len(ohlcv) < limit:
                break

        df = pd.DataFrame(all_ohlcv, columns=['ts','o','h','l','c','v'])
        df['datetime'] = pd.to_datetime(df['ts'], unit='ms')
        df = df[['datetime','c']].rename(columns={'c': ex_id})
        frames.append(df)
    
    final_df = reduce(lambda left, right: pd.merge(left, right, on='datetime', how='outer'), frames)
    final_df.sort_values('datetime', inplace=True)
    final_df.reset_index(drop=True, inplace=True)
    return final_df

def estimate_coint_and_adf(df):

    df = df.dropna(subset=['binance', 'deribit']).copy()

    # 2. Y = binance, X = deribit + intercept
    Y = df['binance']
    X = sm.add_constant(df['deribit'])  # Ajoute l'intercept

    # 3. Rgression OLS => Y = alpha + beta * X
    model = sm.OLS(Y, X).fit()

    alpha = model.params['const']
    beta  = model.params['deribit']

    # 4. Rsidu (epsilon)
    df['epsilon'] = Y - alpha - beta * df['deribit']

    # 5. Test ADF sur epsilon
    adf_result = ts.adfuller(df['epsilon'], autolag='AIC')
    p_value = adf_result[1]

    return alpha, beta, df['epsilon'], p_value, model

Gnration d'un signal bas sur le z-score

Z-Score :

zscore = (spread_smooth mean_spread) / std_spread

Le Z-Score indique de combien dcarts-types le spread courant sloigne de sa moyenne.

  • Un Z-Score proche de 0 signifie que le spread est normal (autour de sa moyenne).
  • Un Z-Score lev (suprieur +2, par exemple) signifie que le spread est trop haut par rapport sa moyenne historique.
  • Un Z-Score bas (infrieur 2) signifie que le spread est trop bas.

window = 500

df['mean_spread'] = df['spread_smooth'].rolling(window).mean()
df['std_spread']  = df['spread_smooth'].rolling(window).std()

# Calcule le Z-score du spread
df['zscore'] = (df['spread_smooth'] - df['mean_spread']) / df['std_spread']

# Exemple simple de stratgie
# Si zscore > +2 => spread "trop haut" => short le spread (short binance, long deribit)
# Si zscore < -2 => spread "trop bas"  => long le spread  (long binance, short deribit)

df['signal'] = 0
df.loc[df['zscore'] >  2, 'signal'] = -1  # short spread
df.loc[df['zscore'] < -2, 'signal'] =  1  # long spread

# Optionnel: fermer la position si le zscore repasse autour de 0
df.loc[df['zscore'].abs() < 0.5, 'signal'] = 0

Signaux de trading :

  • Quand le Z-Score dpasse +2, on considre le spread survalu on prend une position short sur le spread.

(Dans le cas dun spread = prix Binance prix Deribit, tre short sur le spread reviendrait vendre Binance et acheter Deribit.)

  • Quand le Z-Score passe en dessous de 2, on considre le spread sous-valu on prend une position long sur le spread.

(Acheter Binance et vendre Deribit, par exemple.)

  • Pour viter de garder des positions ouvertes quand le signal redevient neutre, on fixe une zone de sortie ou de non-position lorsque le Z-Score se rapproche de 0 (par exemple si abs(zscore) < 0.5).

def plot_trades(df, price_col='spread', signal_col='signal'):
    
    # On va stocker les points (x,y) pour chaque vnement
    enter_long_x, enter_long_y = [], []
    enter_short_x, enter_short_y = [], []
    exit_long_x, exit_long_y = [], []
    exit_short_x, exit_short_y = [], []
    
    # Pour parcourir les changements de signal
    df = df.copy().reset_index(drop=True)
    
    # Signal initial
    prev_sig = df.loc[df.index[0], signal_col]
    
    # Boucle sur chaque range  partir de la 2me
    for i in range(1, len(df)):
        cur_sig = df.loc[i, signal_col]

        if cur_sig != prev_sig:
            # => Le signal vient de changer
            if cur_sig == 1:  
                # Changement vers un long
                enter_long_x.append(df.loc[i, 'datetime'])
                enter_long_y.append(df.loc[i, price_col])
            elif cur_sig == -1:
                # Changement vers un short
                enter_short_x.append(df.loc[i, 'datetime'])
                enter_short_y.append(df.loc[i, price_col])
            else:
                # cur_sig == 0 => On sort d'une position
                if prev_sig == 1:
                    # On sort d'un long
                    exit_long_x.append(df.loc[i, 'datetime'])
                    exit_long_y.append(df.loc[i, price_col])
                elif prev_sig == -1:
                    # On sort d'un short
                    exit_short_x.append(df.loc[i, 'datetime'])
                    exit_short_y.append(df.loc[i, price_col])
        
        prev_sig = cur_sig
    
    # --- Trac du graphique ---
    plt.figure(figsize=(12, 6))
    plt.plot(df['datetime'], df[price_col], label=price_col, color='black')

    # Entres en position
    plt.scatter(enter_long_x, enter_long_y, marker='^', color='green', s=100, label='Enter LONG')
    plt.scatter(enter_short_x, enter_short_y, marker='v', color='green', s=100, label='Enter SHORT')
    # Sorties de position
    plt.scatter(exit_long_x, exit_long_y, marker='v', color='red', s=100, label='Exit LONG')
    plt.scatter(exit_short_x, exit_short_y, marker='^', color='red', s=100, label='Exit SHORT')
    
    plt.title("Backtest avec entres/sorties de position")
    plt.legend()
    plt.show()
